# Grid

## 기능

Grid는 자식 컴포넌트들을 격자 형태로 배치합니다.

만약 자식 컴포넌트들의 크기가 일정하지 않은 경우, 각 칸은 아래 식에 따라 계산됩니다.
- 가로: 해당 열에서 가장 가로 길이가 긴 요소의 크기
- 세로: 해당 행에서 가장 세로 길이가 긴 요소의 크기

## 미리보기

분홍색은 Grid의 외곽을 보여주기 위해 칠한 것이며, 실제 컴포넌트에는 색상이 없습니다. <br />
검은색은 자식 컴포넌트들의 외곽이에요.

## props

Grid는 아래에 적힌 props들을 기본적으로 받을 수 있습니다.
- `rows`: CSS grid-template-rows의 `repeat()`에 들어갈 숫자
- `columns`: CSS grid-template-columns의 `repeat()`에 들어갈 숫자
- `gap`: 하위 요소들 사이의 간격을 나타내는 문자열

추가적으로 `<div>` 태그에 들어갈 수 있는 모든 속성들 역시 prop으로 내려줄 수 있어요.
