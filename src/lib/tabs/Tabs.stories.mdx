# Tabs 컴포넌트

Tabs는 탭을 이용한 콘텐츠의 레이아웃을 구성할 수 있도록 해주는 컴포넌트입니다. 다양한 설정을 통해 유연한 UI를 구성할 수 있습니다.

## 속성

<ArgsTable of={Tabs} />

## 실시간 예시

<Canvas>
  <Story name="컴포넌트">
    {args => (
      <Tabs {...args}>
        <TabList>
          <Tab name={'1'}>
            <h3>first</h3>
          </Tab>
          <Tab name={'2'}>
            <h3>second</h3>
          </Tab>
        </TabList>

        <TabPanelList>
          <TabPanel name={'1'}>

          </TabPanel>
          <TabPanel name={'2'}>
            <div>second Panel</div>
          </TabPanel>
        </TabPanelList>
      </Tabs>
    )}
  </Story>
</Canvas>
